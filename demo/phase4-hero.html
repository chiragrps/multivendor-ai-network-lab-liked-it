Brain. Hands. Undo.

Phase 3 gave the lab a brain. Phase 4 gave it hands — and a way to take them back if something goes wrong. Every config push now flows through RFC 6241 §8.4 confirmed-commit. The Health Gate watches BGP peers, interface state, and alert counts for the entire window. If any signal degrades, the device auto-reverts at the NETCONF timeout. The tool literally cannot leave a router worse than it found it.
