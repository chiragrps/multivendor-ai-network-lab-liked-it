🧠 Layer 4 — AI Orchestration & Phase 3 Tools (8 Modules)
🤖 Pydantic-AI Orchestrator
Auto-classifies prompt → routes to RoutingAgent / ACLAgent / IncidentAgent. Each child returns a Pydantic BaseModel with confidence, evidence, remediation steps. Falls back to deterministic stub when no LLM key.
pydantic_ai_orchestrator.py · 4 BaseModels · Anthropic SDK
🧪 Eval Harness NIKA
10 incident scenarios (BGP, OSPF, MTU, ACL, CVE, intent, perf). Synthesizes NL symptom · invokes agent · scores via keyword overlap + Claude Haiku LLM-as-judge (0–10 each).
eval_harness.py · scenarios.json · LLM judge
🗺️ Vendor Translator
12 canonical tasks × 5 vendors. One symbol "bgp_summary" emits "show bgp summary" (Junos), "show ip bgp summary" (EOS), "vtysh -c '...'" (FRR), or IOS/NX-OS. Destructive tasks gated.
vendor_translator.py · 60 mappings · is_destructive()
🛡️ Auto-Remediation Runbooks
YAML playbooks per fault type (BGP peer down, interface down, OSPF stuck, ACL block). Step types: check / probe / ai / remediate. Destructive steps require approval. Dry-run by default.
runbooks/*.yaml · 4 playbooks
🎯 Intent Verification
Cross-references inventory.json BGP claims vs SuzieQ-parsed observations. Flags claimed_peer_missing, undeclared_peer. Returns drift list + 0–100% intent score.
/api/mv/intent/verify · Drift types · Score
🛣️ Path Trace
BFS over BGP graph + site-LAN edges. Any-to-any 26-device coverage. Edge types: iBGP / eBGP / site-LAN. Renders vendor-colored SVG with hop labels and edge color codes.
/api/mv/path/trace · phase3.js SVG
📜 GAIT Audit Trail
Immutable JSONL append-only log. Every AI action recorded: actor, action, target, prompt, response, tools-called, tokens (in/out), status. Filterable by actor.
gait_audit.py · audit/gait_YYYY-MM-DD.jsonl
